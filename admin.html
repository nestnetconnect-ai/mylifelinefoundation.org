⚠️ Image Upload Fix Required (One-time setup)
Go to: supabase.com → Your Project → Storage → Files → MLF-IMAGES → New Policy
→ Click "For full customization" → Tick INSERT → Target roles: anon → USING expression: true → WITH CHECK: true → Save

OR run this SQL in Supabase → SQL Editor:
CREATE POLICY "allow_all_uploads" ON storage.objects FOR INSERT TO anon WITH CHECK (bucket_id = 'mlf-images');
CREATE POLICY "allow_all_select" ON storage.objects FOR SELECT TO anon USING (bucket_id = 'mlf-images');
Note: Even without this, images save as base64 locally (works fine for small sites)
